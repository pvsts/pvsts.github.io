linux常用命令

· 2023-05-31 ·

详细命令

ls 查看目录
dir 查看目录
cd （文件名） 进入
cd 退一级目录
rm -rf （文件） 删除文件
wget （链接） 下载
ps aux 查看开机进程
touch 1.txt 创建文件
touch 1.txt 2.txt 创建多个
mkdir 1 创建文件夹
rmdir 1 删除文件夹
unzip 1.zip 解压
tar -zvxf 1.tar.gz 解压
mv 移动重命名
cp 1.zip 2.zip 复制
cat frps.ini 查看内容
uname -a 查看架构
whoami 查看ssh登陆账号
:/fun vi里面搜索
sshd & 重启ssh连接
df -h 查看磁盘
free -m 查看空闲
top 运行
chmod 777 * -R 全部子目录及文件权限改为 777
sudo -i root权限
sudo pkill MPOS 杀死全部进程
sudo killall MPOS 杀死全部进程

代码 解释
shutdown 0 直接关机
shutdown -r 0 直接重启
fsck -t ext4 /dev/sda1 修复系统开机进入 initramfs
ip a s 查看ip
ctrl + alt + d 进入桌面
ctrl + alt + t 打开终端
ctrl + alt + F1 进入tty1 f7进入图形化界面
getconf LONG_BIT 查看系统位数
uname -m 查看cpu架构
cp frp_0.52.0_linux_amd64/frpc.ini /tmp 将frpc.ini复制到tmp目录下(/tmp/*)*如果不是路径则重命名文件
chmod +x frpc.ini 给文件所有权限
/tmp/frpc -c /tmp/frpc.ini 启动frpc(如果不是系统路径)./frpc -c ./frpc.ini
tar -zxvf frp_0.52.0_linux_amd64.tar.gz 解压tar.gz
top 查看运行的程序 按m 排序
ps aux grep gnome-terminal
sudo apt-get install apt 安装apt
sudo apt update && sudo apt upgrade -y && sudo apt install x11vnc -y
ls 查看目录
dir 查看目录
cd （文件名） 进入
cd 退一级目录
rm -rf （文件） 删除文件
wget （链接） 下载
ps aux 查看开机进程
touch 1.txt 创建文件
touch 1.txt 2.txt 创建多个
mkdir 1 创建文件夹
rmdir 1 删除文件夹
unzip 1.zip 解压
tar -zvxf 1.tar.gz 解压
mv 移动重命名
cp 1.zip 2.zip 复制
cat frps.ini 查看内容
uname -a 查看架构
whoami 查看ssh登陆账号
:/fun vi里面搜索
sshd & 重启ssh连接
df -h 查看磁盘
free -m 查看空闲
top 运行
chmod 777 * -R 全部子目录及文件权限改为 777
sudo -i root权限
sudo pkill MPOS 杀死全部进程
sudo killall MPOS 杀死全部进程
